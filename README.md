
# Tic Tac Toe Game

A simple Tic Tac Toe game built with HTML, CSS, and JavaScript.  
Players take turns placing X and O on the grid. The first to get 3 in a row wins!

## 🎮 How to Play
1. Open the game in your browser.
2. Player 1 starts with "X".
3. Player 2 plays with "O".
4. First to get 3 in a row (horizontally, vertically, or diagonally) wins.
5. If the board is full and no one wins, it’s a draw.

## ✨ Features
- Scoreboard (tracks Player X, Player O, and Draws)
- Reset button to restart the game
- Displays which player’s turn it is
- Detects win, loss, or draw conditions
- Built using **Factory Functions, Modules, and IIFE** (Immediately Invoked Function Expressions)

## ▶️ How to Run
Just open the `index.html` file in any modern browser.
